# Сборка  
> gcc -DDEBUG -Wall -g main.c bin_tree.c -o res  
> ./res
# Использование
Вводите целые числа, добавля их в двоичное дерево.  
Введите любое выражение отличное от целого числа, вам выдаст сумму элементов дерева  
и предложит проверить наличие элементов в дереве.  
Для завершения программы можно ввести выражение отличное от целого числа, симулировать ситуацию eof (Ctrl+D)
или отправить процессу сигнал SIGINT (Ctrl+C).
