# Сборка  
> gcc -DDEBUG -Wall -g main.c bin_tree.c -o res
> ./res
# Использование
Вводите целые числа, добавля их в двоичное дерево.  
Введите любое выражение отличное от целого числа, вам выдаст сумму элементов дерева  
и предложит проверить наличие элментов в дереве.  
Для завершения программы можно вветси выражение отличное от целого числа, симулировать ситуацию eof (Ctrl+D)
или отправить сигнал SIGINT процессу (Ctrl+C).
